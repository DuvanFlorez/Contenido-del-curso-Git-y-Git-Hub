Volver en el tiempo en nuestro repositorio utilizando reset y checkout

git reset --soft [SHA 1]: elimina los cambios hasta el staging area git reset --mixed [SHA 1]: elimina los cambios hasta el working area git reset --hard [SHA 1]: regresa hasta el commit del [SHA-1] Donde el SHA-1 es el identificador del commit
